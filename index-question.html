Question and Answer

Question 01 : What is DOM and what is the purpose of DOM?

Question 01 Answer :
DOM stands for Document Object Model. DOM is a
programming interface and it is allows us to create, change or
remove elements from the document. We can more dynamic page by
using add dom events to these element. The DOM is a model for web pages.

Question 02 : How will you select HTML elements using DOM? Name the DOM methods.

Question 02 Answer :
There are five ways we can select HTML elements by using DOM. There are...
1=> getElementsByTagName()
2=> getElementsByClassName()
3=> getElementById()
4=> querySelector()
5=> querySelectorAll()

Question 03 : What is event bubble?

Question 03 Answer :
The Event bubbling in JavaScript is a type of
event propagation. It is a procedure where it starts from the
element that triggered the event and then bubbles up to the
ancestor elements in the hierarchy.

Question 04 : What is a callback function and why will you use it?

Question 04 Answer :
A callback function is a function that is
passed as an argument in another function. We should use callback function
because callbacks are used to handle the results of asynchronous
operations in a non blocking manner. When we want one function to execute
only after another function has completed its execution then
we needed use callback functions.
